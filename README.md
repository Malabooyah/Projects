A command-line Blackjack game built in Python that simulates a real game of Blackjack between a player and a dealer. The project uses object-oriented programming to model cards, decks, hands, and gameplay logic while following standard Blackjack rules.

This project was created to strengthen Python fundamentals, OOP design, and debugging skills through a hands-on, interactive application.
